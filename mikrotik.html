RouterOS helper

MikroTik Script Generator

Инструмент для быстрой генерации базовых команд RouterOS: NAT, DNS и port forwarding с проверкой параметров и выгрузкой в .rsc.

Перед применением проверь

  • IP-адреса и подсети соответствуют твоей сети
  • WAN interface list указан корректно
  • dst-address совпадает с внешней схемой доступа
  • Hairpin NAT нужен только если сервис открывается из LAN по внешнему адресу

NAT

Генерирует базовое masquerade-правило для локальной подсети с быстрым выводом команд для RouterOS.

DNS

Собирает конфигурацию DNS, умеет включать redirect запросов с LAN и блокировку запросов с WAN.

Port Forward

Формирует правила dstnat и при необходимости добавляет hairpin NAT для внутренних клиентов.

Генератор конфигураций

Выбери сценарий, заполни параметры и сразу получи готовые команды для RouterOS.

Базовый masquerade для локальной сети

Подходит для стандартного сценария, когда роутер выпускает локальную подсеть в интернет через srcnat.

Используй CIDR-нотацию, например 192.168.88.0/24.
Используется как ориентир для схемы и комментариев.
