# Портфолио DmiRials

Сайт-портфолио для публикации работ, проектов и демонстрации навыков.

## Структура сайта

- `index.html` - главная страница
- `style.css` - стили оформления
- `script.js` - JavaScript функционал

## Как добавить новую работу

### Вручную в HTML

Откройте `index.html` и добавьте новый блок в секцию `#portfolioGrid`:

```html
<div class="portfolio-item">
    <div class="portfolio-image">
        <img src="путь/к/изображению.jpg" alt="Название">
    </div>
    <div class="portfolio-content">
        <h3>Название проекта</h3>
        <p>Описание проекта</p>
        <div class="tags">
            <span class="tag">Тег1</span>
            <span class="tag">Тег2</span>
        </div>
    </div>
</div>
```

### Через JavaScript

Используйте функцию `addPortfolioItem()` в `script.js`:

```javascript
addPortfolioItem(
    'Название проекта',
    'Описание проекта',
    'путь/к/изображению.jpg',
    ['Тег1', 'Тег2']
);
```

## Размещение изображений

1. Создайте папку `images` в репозитории
2. Загрузите изображения
3. Используйте путь: `images/название_файла.jpg`

## Кодировка

Все файлы используют UTF-8 для корректного отображения русских символов.

## Настройка

Отредактируйте файлы для персонализации:
- Измените имя и описание в `index.html`
- Добавьте свои навыки в секции "О мне"
- Замените примеры работ на реальные проекты
- Обновите ссылки в секции "Контакты"
